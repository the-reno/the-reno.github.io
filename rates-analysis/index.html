Corrected logic

  • Every eligible curve date is a valid daily investment start.
  • The actual maturity is the latest curve date on or before the tenor target.
  • The same-tenor maturity rate is compared with the same-tenor start rate.
  • Reset volatility remains in basis points without annualization or tenor normalization.
  • Historical-rate charts retain actual Excel dates and chronological time-scale axes.
  • The efficient frontier uses aligned month-end strategy returns and remains separate from reinvestment reset risk.
  • Every frontier point includes a rank, segment, description, allocation summary, WAM and liquidity profile.

Use in Excel

  1. Download the complete package or both BAS files above.
  2. Open a blank macro-enabled workbook.
  3. Import Rates_Analysis_Structure.bas through the VBA editor.
  4. Import Rates_Analysis_Model.bas.
  5. Run CreateRatesAnalysisStructure.
  6. Enter the inputs and paste the curve with headers Date | ON | 1M | 2M | 3M | 6M.
  7. Run BuildRatesAnalysisModel.
Rates are entered as percentage points: 4.31 means 4.31%. The rolling strategy uses ACT/360. The two modules must be imported into the same workbook.
