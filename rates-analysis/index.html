What changed

  • Every eligible curve date is a real start-date scenario for every tenor.
  • The model compares the same-tenor rate at the start with the rate available at actual maturity.
  • Reset volatility remains in basis points without tenor normalization or annualization.
  • Every efficient-frontier point has a rank, segment, description, allocation summary, WAM and liquidity profile.

Use in Excel

  1. Download the version 2 VBA module above.
  2. Open a blank macro-enabled workbook.
  3. Import Rates_Analysis_Final_v2.bas through the VBA editor.
  4. Run CreateBlankRatesModel.
  5. Enter the inputs and paste the curve with headers Date | ON | 1M | 2M | 3M | 6M.
  6. Run BuildRatesAnalysisModel.
Rates are entered as percentage points: 1.54 means 1.54%. The model uses ACT/360, reinvests interest and applies the latest prior curve date when a target date is unavailable.
