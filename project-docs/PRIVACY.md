# Privacy maintenance

The public policy is `docs/privacy/index.html`.

Current website behavior:

- no first-party analytics
- no advertising pixels
- no cookies or browser storage
- no forms, accounts, or payments
- interactive inputs remain in the visitor's browser
- GitHub Pages provides hosting and may retain infrastructure logs
- Google Fonts is used on public pages
- cdnjs supplies KaTeX on the dice page

Before adding analytics, embeds, forms, comments, videos, maps, external scripts, or user accounts:

1. identify what data is collected and why;
2. minimize collection and retention;
3. update the public privacy policy;
4. update Content Security Policy directives;
5. obtain consent where legally required; and
6. test that the page still works when optional tracking is blocked.

Never describe the site as collecting no data if a new provider, form, or analytics service changes that fact.
