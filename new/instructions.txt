================================================================================
 DEPOSIT LADDER OPTIMISER  ·  VBA engine        (folder: depo new)
 Fixed-rate time deposits · ON / 1M / 2M / 3M · easy input, precise output
================================================================================

WHAT'S NEW IN THIS VERSION
  - modSetup: a macro that BUILDS THE WHOLE WORKBOOK for you (sheets, headers,
    named inputs, and a Home control panel with buttons). No manual setup.
  - Income-volatility metric baked into the engine (the risk-management number
    for comparing mixes: how smooth the earned book-yield is).
  - modDashboard: one "Results" sheet with the valuable read-out.
  - One consolidated file (DepositLadder_AllModules.txt) holding every module.

--------------------------------------------------------------------------------
INSTALL  (about a minute)
--------------------------------------------------------------------------------
  1. Open a blank workbook -> Save As -> Excel Macro-Enabled Workbook (.xlsm).
  2. Alt+F11 -> right-click the project -> Import File... and import the ten
     .bas modules (or paste the single DepositLadder_AllModules.txt as modules).
  3. Back in Excel: Alt+F8 -> run  BuildTemplate.
     That creates Home, Config, Curve and Audit and a button panel.

--------------------------------------------------------------------------------
USE  (three steps)
--------------------------------------------------------------------------------
  1. (optional) click  LOAD SAMPLE  to drop in a test curve and try it now.
  2. On the  Curve  sheet, paste your daily curve:
        col B = date,  C..F = ON / 1M / 2M / 3M rates in %.
  3. Click  RUN ALL.

--------------------------------------------------------------------------------
INPUTS  (that's all you touch)
--------------------------------------------------------------------------------
  Curve   your daily rates (blue cells).
  Config  notional, day-count (360), grid step, tenor days. Sensible defaults.
  Audit   four weights for a single mix you want the cash-flow ledger on.

--------------------------------------------------------------------------------
OUTPUTS
--------------------------------------------------------------------------------
  Results       best-carry mix, smoothest-income mix, per-tenor return, WAM,
                income vol, and a plain-language read-out.
  VBA_Frontier  every mix: carry vs liquidity (WAM) and income volatility.
  VBA_Window    rolling 1-year window - which tenor won, over time.
  VBA_Scenario  forward reinvestment risk under +/- curve shocks.
  VBA_Audit     granular ledger for one mix: every PLACE / MATURE / REINVEST.

--------------------------------------------------------------------------------
MODULES
--------------------------------------------------------------------------------
  modSetup      BuildTemplate, LoadSampleCurve  (creates the workbook)
  modConfig     TConfig holder + per-tenor scenario shift
  modCurve      loads curve; RateAt / DaysBetween / NextMatIdx
  modEngine     SimulateStrategy: sleeve-roll ladder, ACT/360, income vol, audit
  modFrontier   weight-grid frontier + EnsureSheet + MakeScatter
  modScenario   curve-shock scenarios -> forward reinvestment risk
  modWindow     full-period + rolling 1-year window
  modDashboard  the Results read-out
  modAudit      single-mix cash-flow ledger
  modMain       button entry points (RunEverything, etc.)

Model note: each tenor is one rolling sleeve, reinvested in full at maturity
(true hold-to-maturity; no early break). Weights drift only via differential
compounding. Income vol = volatility of the ~monthly change in earned
book-yield - lower means a smoother, more predictable income stream.
