// 01 · The Experiment

Step through the pattern

一
// start here
Every number, a dot

Each number n is placed n steps from the center, rotated by n radians. Distance and direction from the same value.

二
// the geometry
Angles wrap like a clock

After one full spin you face the same way again. Some step-sizes nearly repeat — dots pile into lanes. Rays emerge from nothing.

三
// primes only
Primes avoid the crowd

Keep only primes. Some rays go dark — because those lanes belong to numbers divisible by 2, 3, or 5. Primes refuse to live there.

四
// one ray
A single family, revealed

The glowing ray holds every number sharing the same remainder mod 44. Adding 44 to n spins by almost 7 full turns — landing in the same direction.
