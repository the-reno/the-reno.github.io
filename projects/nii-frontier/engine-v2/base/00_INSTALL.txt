NII ENGINE v2 — VBA MODULES
============================
Four modules. Paste each into the VBA editor (Alt+F11).

  mRegistry.txt    Insert > Module          rename to mRegistry
  cRatesCurve.txt  Insert > Class Module    rename to cRatesCurve
  mEngine.txt      Insert > Module          rename to mEngine
  mDeposit.txt     Insert > Module          rename to mDeposit

After pasting all four: File > Save As > Macro-Enabled Workbook (.xlsm)
Then: Debug > Compile VBAProject  (must compile clean before use)

CURVE FUNCTIONS (type in cells)
  =RBuildCurve(name, start, end, sofr, fedRange, holidayRange)
  =RCurveRate(curveCell, date)
  =RAccrue(start, end, amount, "SIMPLE"|"COMPOUND", curveCell)
  =RSwapLeg(start, end, notional, fixed, curveCell, "FIXED"|"FLOAT"|"NET")

DEPOSIT MACRO (Alt+F8 > BuildDepositLedger)
  Reads from INPUTS sheet:
    B4  Start date
    B5  Horizon in years (0.5 / 1 / 1.5 / 2)
    B6  ON allocation  ($mm)
    B7  1M allocation  ($mm)
    B8  2M allocation  ($mm)
    B9  3M allocation  ($mm)
    A14:A30  Holiday dates (one per row)
  Writes every deposit event to LEDGER sheet:
    PlaceDate | MaturityDate | Bucket | Event | Notional($mm) |
    Tenor(days) | RateDate | Rate% | Interest($mm)
  Rate% and Interest are left blank. Wire them after building a curve:
    H2: =RCurveRate($B$curveCell, G2)
    I2: =RAccrue(A2, B2, E2, "SIMPLE", $B$curveCell)

RULE: pass the curve cell as a CELL reference ($B$6), never typed text.
