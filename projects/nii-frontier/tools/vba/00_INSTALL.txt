NII ENGINE - VBA MODULES (install order)
=========================================
Workbook: NII_Engine_Example_v1.0.xlsx (ships macro-free; #NAME? until setup)

  01_mInterestCalc.txt   standard Module   (5 accrual primitives - signed off)
  02_mTests.txt          standard Module   (Alt+F8 > TestInterestEngine)
  03_mRegistry.txt       standard Module   (name -> object cache)
  04_cCalendar.txt       CLASS Module      (Insert > Class Module, rename cCalendar)
  05_cStepCurve.txt      CLASS Module      (Insert > Class Module, rename cStepCurve)
  06_mSheetAPI.txt       standard Module   (FED_HOL, MAKE_CURVE, ON_INTEREST,
                                            PERIOD_INTEREST, SWAP_NET)

Paste route: skip each file's first 'Attribute VB_Name' line and rename the
module to match. Then save as .xlsm and press Ctrl+Alt+F9.

Conventions (binding): handles flow cell-to-cell, never typed strings.
Growable inputs are Excel Tables. See ARCHITECTURE.txt.
